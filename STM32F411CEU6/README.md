# STM32F411CEU6 Project (cshttt)

This is an STM32CubeIDE project for the STM32F411CEU6 microcontroller.

## Description
This repository contains the source code, configurations, and generated files for the `cshttt` project, developed and exported from STM32CubeIDE.

## Hardware Required
- STM32F411CEU6 (Black Pill or similar development board)
- ST-Link V2 or similar programmer

## Software Required
- STM32CubeIDE

## Getting Started
1. Clone this repository.
2. Open STM32CubeIDE.
3. Import the project: `File` -> `Import...` -> `General` -> `Existing Projects into Workspace`.
4. Select the project directory and click `Finish`.
5. Build and flash the project to your board.
